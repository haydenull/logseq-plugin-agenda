# 命令及快捷键

# 命令

打开命令面板:

  • MacOS: cmd + shift + p
  • Windows: ctrl + shift + p

支持命令面板命令:

  • show agenda: 打开插件

# block 圆点右键菜单

  • Agenda: Modify Schedule: 编辑事件

# 斜杠命令

  • Agenda: Modify Schedule: 编辑事件
  • Agenda: Insert Today's Task: 插入今天的事件
  • Agenda: Insert Task List: 插入事件列表

# 快捷键

  • Esc: 关闭插件
